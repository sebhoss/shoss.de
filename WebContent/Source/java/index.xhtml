Dies sind Java Programme, die während des Studiums geschrieben wurden.
  • MyVector
    Die Klasse MyVector bietet die Möglichkeit einen Vektor mit fixer Länge zu erstellen, die einzelnen Elemente auszulesen, sie zu verändern, den Vektor mit einem anderen zu addieren, den Vektor mit einem Skalar zu multiplizieren und das Skalarprodukt aus 2 Vektoren zu bilden. Enthalten ist ebenfalls eine Testklasse, die alle Methoden prüft.
  • DoppelStack
    Die Klasse DoppelStack speichert 2 Stacks innerhalb eines Arrays und bietet die für einen Stack üblichen Operationen an. (push,pop,top,isEmpty,create). Dabei ist wieder eine Testklasse, die alle Methoden prüft.
  • Fibonacci
    Die Klasse Fibonacci stellt 3 unterschiedliche Methoden zur Berechnung der Fibonaccizahlen bereit: Rekursiv, iterativ und modular. Die modulare Methode funktioniert nach dem Chinesischem Primzahlsieb, wobei die moduli 999,1000 und 1001 verwendet werden. Die Klasse wird durch eine JUnit Testklasse überprüft und fängt eventuelle falsch-Eingaben durch eine Exception ab.
  • ThreeIntegers
    Die Klasse bietet die Möglichkeit 3 Integer Zahlen in eine einzige Zahl umzuwandeln und diese wieder zurück in 3 Integer Zahlen zurück zuwandeln. Die Klasse wird wiederrum durch eine kleine JUnit Testklasse überprüft.
  • ArraySearch
    Die Klasse bietet die Möglichkeit eine m x n Matrix nach einem Element zu durchsuchen. m steht dabei für die Anzahl an Spalten, n für die Anzahl an Zeilen. (JUnit Test inklusive)
  • MyMatrix
    Speichert eine antisymmetrische Matrix und gibt deren Werte wieder.
  • ComputePairsForSum
    Sucht in einem Array nach Paaren die zusammen eine festzulegende Summe bilden und gibt diese dann geordnet wieder.
  • Ant Beispiel buildfile
    Ein kleine Beispiel Ant Beispiel. Führt die 4 Targets - build, clean, package, run - aus.
